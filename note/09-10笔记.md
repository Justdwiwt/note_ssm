## mybatis对单值传递的操作

> `#{参数是任意的，但是不能不写}`

``` xml
    <select id="selectUser" resultType="User" parameterType="long">
        SELECT *
        FROM user
        WHERE id = #{id}
    </select>
```

* 使用`id`单条件查询和单条件删除时，#{}
* 使用`update`语句和`select`全部查询语句
